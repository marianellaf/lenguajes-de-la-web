Ejercicio: Lenguajes de la web

INSTRUCCIONES

  1. Crear un carpeta en tus documentos que se llame laboratoria (Si es que ya la creo no es necesario repetir el proceso).
  2. Dentro de la carpeta laboratoria, crea otra carpeta llamada Lenguajes de la Web
  3. Descarga el zip que se encuentra en la sección de Descarga y guardalo en la carpeta creada (Lenguajes de la web).
  4. Extrae el contenido del ZIP en la misma ruta (http://7zip-es.updatestar.com/)
  5. Dentro del ZIP encontrarán una carpeta de img, un .ico y un file index.html

Parte 1

Con los archivos proporcionados crear la estructura HTML necesaria para que se vea igual que en la imagen

Parte 2

Una vez que la estructura HTML está lista realice los siguientes cambios:

Para el título cambie los siguientes elementos:

  • color: red
  • peso de la letra: 500
  • letra: Arial
  • tamaño de letra: 20px

Para cada lenguaje:

  • Que la imagen tenga un borde (color: blue, de 1px y sólido)
  • Que el párrafo sea (color: yellow, tamaño de letra 14px)

Para el h2 sea:

  • color: black
  • tamaño de letra 16px

Parte 3

Traslada todos los estilos de la página del “Lenguajes de la web” al &lt;head&gt;

Parte 4

Traslada todos los estilos de la página del “Lenguajes de la web” a un nuevo archivo css (main.css) y conecta el HTML con el CSS
